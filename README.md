# Instalação:

Instale as dependências do Tailwind CSS:
```bash
npm install -D tailwindcss
```

Crie o arquivo de configuração do Tailwind:

```bash
npx tailwindcss init
```

Instalar o Tailwind:

```bash
npm install @tailwindui/react
```


## Executar:
```bash
$ npm install
$ npm install react-router-dom
$ npm start
```
